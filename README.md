## Tarea de Desarrollo Basado en Plataformas

### Para la tarea se necesita crear un venv

Para crear el venv se usan los siguientes comandos:
`python3 -m venv venv`

Y luego para activarlo:
`source venv/bin/activate`

### Se deben instalar las siguientes librerías

- sqlalchemy
- faker

`pip install sqlalchemy faker`
